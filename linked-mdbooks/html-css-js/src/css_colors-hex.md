# HEX

Each of the 6 HEX-values can be [0-9][A-F], which make up to 16 possibilities: `(16x16, 16x16, 16x16) = (0-255, 0-255, 0-255)`. 

### DEC -> HEX
- `rgb(0,0,0) = #000000`
- `rgb(255,255,255) = #FFFFFF`

### Abbreviations

- `#000000 = #000`
- `#FFFFFF = #FFF`
- `#AA33BB = #A3B`
