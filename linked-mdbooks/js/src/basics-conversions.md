# Type-Conversions

```js
// convert not-numeric values to numeric ones
+3     // 3
+'3'   // 3
+true  // 1
+false // 0
+null  // 0

// unary plus/negation operator
var str = "4"	// "4" (String)
var num = +str 	// -4 (Number)

// TYPE CONVERSIONS
let num1 = parseInt("1");  // converts string to number (float or int)
parseInt(string, radix);   // radix=2 -> converts from binary
```
